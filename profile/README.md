## What is UgCore?
 
UgCore is a FiveM server framework designed to give developers a solid, well-structured foundation without the bloat. It handles the essentials — player management, character data, permissions, and cross-resource exports — so you can focus on building the server you actually want.
 
Built with **oxmysql** in mind, and designed to play nicely with the modern FiveM ecosystem.
